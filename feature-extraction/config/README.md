# Extractor configuration files

## projects.config

This file has the format:

<Repository Name 1>

<Project Name 1>

.

.

.

<Project Name N>


<Repository Name 2>

<Project Name 1>

.

.

.

<Project Name N>


To exclude a repository or a project add a # before the specific line.

*Example:*
1) Exclude defects4j repository


#defects4j

Lang

Math

...

2) Exclude Lang project


defectss4j

#Lang

Math
